Daily Tutorial: Getting Started with Programming

If you want to start programming, you need to get started with the basics. Important concepts are:

  • Events: When the user does something
  • Loops: Doing repeated tasks more efficiently
  • Conditionals: When something happens
  • Functions: piece of coode that can be reused in one line
  • Variables: computer's memory
  • Lists: like Variables, but supportive with multiple values

A good website to get you used to the concepts is Scratch, a block based coding engine.
